# 🎮 XR Storyroom 遊戲操作與熱鍵指南 (Operation Manual)

**最後更新時間：** 2026-04-17
**用途：** 詳細記錄專案中所有的操作按鍵、除錯熱鍵以及導演模式指令。

---

## 🏃 1. 玩家基礎操作 (Player Controls)

在進入遊戲且鎖定滑鼠 (游標消失) 後，可使用以下按鍵進行基礎遊玩：

| 按鍵 (Key) | 動作 (Action) | 說明 (Description) |
| --- | --- | --- |
| **滑鼠移動 (Mouse)** | 視角轉動 | 轉動攝影機，觀看四周環境。 |
| **W, A, S, D** | 移動 (平移) | W (前進), S (後退), A (左平移), D (右平移)。 |
| **長按滑鼠左鍵** | 念動力 (Telekinesis) | 抓取並拖曳場景中具有物理互動屬性的物件 (例如方塊)。 |
| **Space 或 Esc** | 跳過動畫 / 釋放游標 | 於過場動畫(Cinematic)時按空白鍵可跳過；遊玩中按 Esc 呼叫主選單。 |

---

## 🎥 2. 視角與攝影機切換 (Camera Modes)

遊戲在第三人稱視角下提供不同的攝影機運鏡模式，適合觀影或遊玩：

| 按鍵 (Key) | 視角模式 (Mode) | 說明 (Description) |
| --- | --- | --- |
| **1** | 跟隨視角 (OTS) | 預設視角，位於玩家角色後方（越肩視角）。 |
| **2** | 廣角視角 (Wide) | 將攝影機高度拉高且退後，適合觀賞大場景。 |
| **3** | 主觀視角 (First-Person) | 隱藏狐狸角色，切換為第一人稱超廣角 (Fisheye) 視角。 |

---

## 🛠️ 3. 開發者除錯與輔助系統 (Developer Debug)

方便快速觀看系統效能或場景設定：

| 按鍵 (Key) | 功能 (Function) | 說明 (Description) |
| --- | --- | --- |
| **F3 或 \` (波浪鍵)** | 開啟 Debug 儀表板 | 開啟/關閉位於右上角的綠色除錯面版 (顯示 FPS、Draw Calls、狀態等)。 |
| **UI 直接點選** | 音效開關 (🔊) | 切換全域音效靜音/開啟。 |
| **UI 直接點選** | 開啟/關閉 Bloom 特效 | 於首頁選單內可切換畫面泛光特效，測試效能差異。 |

---

## 🎬 4. 導演分鏡編輯器 (Director Editor Mode)

專門用來錄製過場動畫序列與開發關卡的強大模式：

* **開啟方式：** 點選畫面左上角的「🎬 編輯器模式」按鈕。
* **Noclip 自由飛行：** 勾選面板上的「Noclip」，此時角色將無視重力與碰撞。
* **Noclip 操作熱鍵：**
  * **W A S D**：水平飛移。
  * **Q**：向上攀升 (上升)。
  * **E**：向下沉潛 (下降)。
* **座標紀錄流程：**
  1. 飛到理想的起點，按下「📍 存起點」。
  2. 飛到理想的終點，輸入秒數 (Duration)，按下「🏁 存終點」。
  3. 輸入對白後，點選「➕ 新增至腳本」。
  4. 最後點擊「匯出程式碼 (Export)」，即可獲取供 `narrative.js` 使用的陣列設定。
