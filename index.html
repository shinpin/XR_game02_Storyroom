🎮 操作與熱鍵指南 (Operation Manual)

🏃 玩家基礎操作

  • 滑鼠移動：視角轉動
  • W, A, S, D：前後左右平移移動
  • 長按滑鼠左鍵：念動力 (抓取並拖曳物理物件)
  • Space / Esc：跳過過場動畫 / 呼叫主選單釋放游標

🎥 視角與系統操作

  • 1 / 2 / 3：切換攝影機 (跟隨 / 廣角 / 第一人稱 主觀視角)
  • F3 或 \` ：開啟 / 關閉開發者效能除錯面板 (Debug Mode)

🎬 導演模式 (Noclip)

  • 點擊左上角「編輯器模式」並勾選 Noclip 啟用
  • Q / E：上升 / 下降 (自由飛行)
